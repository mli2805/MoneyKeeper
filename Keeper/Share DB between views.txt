

  [Export(typeof(KeeperDb)), PartCreationPolicy(CreationPolicy.Shared)]
  public class KeeperDb : DbContext


Класс БД. Инстанс данного класса не создается у меня в коде
т.к. класс помечен Export(typeof(KeeperDb)) - инстанс его будет
создан MEFом, а т.к. класс помечен PartCreationPolicy(CreationPolicy.Shared),
то этот инстанс будет разделен между всеми классами, которые 
хотят работать с БД и определили у себя св-во

    [Import]
    public KeeperDb Db { get; set; }

или

    public KeeperDb Db { get { return IoC.Get<KeeperDb>(); } }
    public static KeeperDb Db { get { return IoC.Get<KeeperDb>(); } }

- это два способа потребовать у MEFа инстанс БД. Второй способ лучше?!
